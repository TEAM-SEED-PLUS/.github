# SEED+

> **점포형 창업자를 위한 AI 기반 입지 의사결정 플랫폼**

창업을 준비할 때 가장 어려운 질문 —
**"이 자리에 가게를 열면 살아남을 수 있을까?"**

SEED+는 흩어진 창업 정보를 하나로 통합해, 누구나 데이터 기반으로 창업 의사결정을 내릴 수 있도록 지원합니다.

---

## 왜 SEED+인가?

소상공인 5년 내 누적 폐업률은 약 60~70%에 달합니다.
폐업의 주요 원인은 기술력 부족이 아니라 **창업 전 정보 비대칭**입니다.

| 기존 창업자의 현실 | SEED+로 달라지는 것 |
|---|---|
| 정부 사이트에서 데이터 직접 수집 후 혼자 해석 | 클릭 한 번으로 종합 점수 확인 |
| 권리금 적정 시세를 "감"으로 판단 | 전국 실거래 권리금 DB 즉시 조회 |
| 내 가게 생존 가능성을 알 수 없음 | AI 기반 3년·5년 생존율 % 예측 |
| 변호사·세무사·중개사 각각 개별 연락 | 플랫폼 내에서 전문가 즉시 연결 |

---

## 주요 기능

### 1. Property Score — 입지 종합 평가 지수
유동인구, 매출 추정, 경쟁 밀도, 권리금, 임대료, 생활편의도 **6개 변수**를 가중 평균하여 **100점 만점 단일 점수**로 출력합니다.

> 예시: `여의도 A 상가 = 77점 / 동일 업종 상위 23%`

비전문가도 한눈에 판단 가능한 결론을 제공합니다.

---

### 2. AI 생존율 계산기
업종, 위치, 권리금, 경쟁사 수, 유동인구를 입력하면 **3년·5년 생존율(%)** 을 예측하고, 유사 조건의 성공 사례 3건을 함께 추천합니다.

> "내가 이 자리에서 살아남을 수 있을까?" → 숫자로 직접 확인

---

### 3. 실거래 권리금·임대료 DB
전국 공인중개사 파트너십을 통해 수집한 **실거래 권리금·보증금·임대료 데이터**를 제공합니다. 현재 국내 어떤 서비스에서도 제공하지 않는 업계 유일 데이터입니다.

> 6개월 주기 갱신 / Property Score 산출 변수로 통합 활용

---

### 4. 전문가 즉시 연결
플랫폼 내에서 **공인중개사 · 세무회계사 · 법무법인**에 즉시 연결됩니다. 상담부터 계약까지 별도 서비스 이동 없이 처리됩니다.

- 공인중개사: 매물 검토, 계약서 확인
- 세무회계사: 사업자 등록, 절세 전략
- 법무법인: 권리금 계약서, 임대차 법률 검토

---

### 5. 상권날씨 콘텐츠
주 1회, 실제 데이터 기반의 상권 분석 콘텐츠를 제공합니다.

> 예시: `"실전 데이터로 본 여의도 카페 3년 생존율"`
